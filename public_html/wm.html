The UMich Smartphone Network Test 1.05 Beta

Windows Mobile application description

This is an application written in C++. In this application, we implemented basic functions such as Ping, Traceroute, and Ipconfig, so that you can know basic network properties of your phone, such as IP address, Gateway, Ping round trip time, DNS lookup time. Besides these information, our application can also help users get more fine-grained information. For example, we help users tell whether their carrier differentiates any kind of application traffic. To achieve this, we emulate the traffic of BitTorrent. Also, our application attemps to detect the existence of HTTP proxy caching and compression.

We are from the RobustNet Group from the Computer Science & Engineering in the University of Michigan.

Installation on Windows Mobile

Solution 1:
1. Download MNet.cab to your phone.
2. Click MNet.CAB and install.
3. Run Program Files/MNet/mnet.exe


Solution 2:
1. Download MNet.zip to your desktop.
2. Extract the compressed file to a folder like "mnet".
3. Copy the whole folder to your phone.
4. Run mnet.exe
