The UMich Smartphone Network Test 1.05 Beta

iPhone application description

This is an application written in Objective-C. This application runs several tests to tell the properties of your network, for example what network are you using, is you phone behing a network address translator, what is your average DNS lookup latency. This application also tells you if your bit torrent traffic is blocked based on timing out while receiving bit torrent data. This applications also tells the uplink and downlink speed of network.

We are from the RobustNet Group from the Computer Science & Engineering in the University of Michigan.
